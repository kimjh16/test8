# 피보나치 수열을 정수 N에 대해 순환적 방법과 재귀적 방법으로 계산하는 코드를 분석,비교한다

# 재귀적 방법: 재귀적 방법은 함수가 자기 자신을 반복적으로 호출하여 값을 계산한다. 이 방법으로 N값을 45로 정하고 프로파일링을 할 시 실행 시간은 약 26초 정도 나오고 cpu사용량이 일정하게 사용중인것으로 나온다 이는 N이 증가할수록 재귀적 방법은 수행시간이 급격하게 늘어나며 따라서 시간복잡도는 O(2^n)이 된다. 

# 순환적 방법: 순환적 방법은 반복문을 통해 이전 두 값을 더해 차례대로 계산하며 계산된 값을 사용하므로 재귀적 방법과 다르게 같은 값을 반복적으로 호출하여 계산하지 않는다. 이 방법으로 똑같이 N값을 45로 정하고 프로파일링 할 시 실행 시간은 약 5초 정도이고 cpu 사용량은 거의 보이지 않는다 따라서 순환적 방법은 N이 증가하면 비교적 수행시간이 일정하게 증가하므로 시간복잡도는 O(n)이 된다.

# 결론: 재귀적 방법은 함수를 호출하는 과정에서 지수적인 시간 복잡도인  O(2^n)을 가지는 반면, 순환적 방법은 이전에 계산한 값을  가지고 반복문으로 결과를 내므로 선형적인 시간 복잡도인 O(n)을 가지기 때문에 큰 값을 처리할 때에는 성능 저하가 심한 재귀적 방법보다는 순환적 방법이 좋은것을 알 수 있다.
